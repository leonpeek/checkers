# Checkers
This project implements a 8x8 [checkers](https://en.wikipedia.org/wiki/Draughts) game. The function of this program is limited, only support local chess, do not support the man-machine chess and chess in network. The initial goal of this project is to provide a basic framework for testing machine learning algorithms. The project is currently in its infancy. Â
